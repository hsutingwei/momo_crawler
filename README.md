# Momo 商品爬蟲與數據處理專案

## 專案簡介
本專案旨在從 Momo 購物網站爬取商品資訊與用戶評論，並對數據進行預處理與斷詞分析，為後續數據分析與應用做準備。

## 目錄結構
```
.
├── crawler.py      # 負責爬取 Momo 網站的商品資訊與評論
├── preprocess.py   # 負責數據預處理（繁簡轉換、去重留言等）
├── seg.py         # 進行斷詞處理
└── README.md      # 本文件
```

## 環境需求
請確保您的開發環境已安裝以下套件：

```bash
pip install requests beautifulsoup4 pandas opencc ckip-transformers
```

如果要使用 GPU 加速 CKIP Transformers，請先確認您的環境支援 CUDA，並安裝相應的 CUDA 和 cuDNN 版本。

### 確認 GPU 支援
執行以下命令確認是否有可用的 GPU：
```bash
python -c "import torch; print(torch.cuda.is_available())"
```
如果返回 `True`，表示您的環境支援 GPU。

### 安裝 CUDA 和 cuDNN（適用於 PyTorch）
請根據您的顯卡型號與 PyTorch 版本，從 [NVIDIA 官網](https://developer.nvidia.com/cuda-toolkit) 下載對應版本的 CUDA 和 cuDNN。

## 使用方法
### 1. 爬取 Momo 商品與評論
執行 `crawler.py` 以開始爬取數據：
```bash
python crawler.py
```
- **模擬滾輪下滑**：由於 Momo 頁面採用動態加載技術，需要模擬滾輪滾動以觸發評論區的加載。
- **換頁取得留言**：每個商品的評論分佈在多個頁面，爬蟲會自動點擊下一頁按鈕，確保能獲取完整的評論數據。
- **設定隨機睡眠時間**：為了避免被網站反爬機制封鎖，爬蟲會在請求之間隨機暫停一段時間，模擬真實用戶行為。
- **儲存數據**：將商品資訊與評論內容存入本地 CSV 文件，供後續分析使用。

### 2. 數據預處理
執行 `preprocess.py` 以清理與轉換數據：
```bash
python preprocess.py
```
該腳本負責：
- **全形轉半形**：由於 Momo 網站的評論可能混雜全形字體，需要轉換為統一格式，方便後續處理與分析。
- **進行繁簡轉換**：網站的評論可能混雜繁體與簡體中文，為確保數據一致性，需要轉換為統一格式，方便後續處理與分析。
- **移除重複評論**：某些評論可能會重複出現，影響分析結果，因此需要去重，以保證數據的準確性。
- **清洗數據**：移除無意義符號、HTML 標籤、特殊字元等，確保數據乾淨，以便後續 NLP 處理。

### 3. 斷詞處理
執行 `seg.py` 以對評論數據進行斷詞：
```bash
python seg.py
```
- **使用 CKIP Transformers 進行斷詞**：基於 CKIP 的 BERT 模型進行詞性標註與分詞，以提升分詞準確度。
- **去除單字詞**：由於單字詞在中文 NLP 分析中往往無意義，會被移除以降低噪音。
- **根據詞性過濾雜詞**：不同於傳統停用詞表方式，該腳本透過 CKIP 提供的詞性標註來去除無意義詞，例如助詞、語助詞等，而非單純基於停用詞表篩選。
